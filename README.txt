TEO (skracenica za "TErapija euthyrOxom") je program za slijepe/slabovidne osobe koje boluju od hipotireoze, 
poremecaja rada stitne zlijezde, i svakodnevno trose euthyrox tablete. Euthyrox tablete dolaze u "okruglim" 
iznosima miligrama, pa sam i program tome prilagodio. Da nebi svako jutro slijepe (ali i one koje dobro vide,ali su 
mozda ljene gledati na kalendar) osobe morale misliti koji je datum/dan u tjednu,onda o 
tome vodi racuna baza podataka temeljena na inicijalnim unosima.
 
Graficko sucelje napravio sam koristeci Tkinter i programski jezik Python 2.7. Za rad s relacijskom bazom podataka koristio
sam SQLite3 modul. 
Glas koji cita doze koje je potrebno popiti za konkretan dan, cita zahvaljujuci mbrolinoj bazi hrvatskih difona koju sam skinuo s mbroline sluzbene stranice.
Mbrolin sintetizator (http://tcts.fpms.ac.be/synthesis/mbrola.html ) je potrebno instalirati, inace program ne radi. 
Isto tako potrebno je imati instalirani python shell s obzirom da nisam kompajlao u izvršnu datoteku (budem
u buducnosti mozda) pa je za pokretanje
potrebno imati instaliran shell.
